Panduan Penggunaan Aplikasi : 
1. Aplikasi dapat dijalankan di lokal pengguna. IDE yang disarankan untuk menjalankan aplikasi adalah eclipse/netbeans.
2. Pastikan module dan dependensi yang diperlukan telah terinstall dalam komputer pengguna termasuk Enviroment Variable yang telah
di setting sebelumnya.
3. Pada aplikasi ini memiliki fungsionalitas push notification menggunakan Firebase Cloud Messaging.
4. Pada sisi front end maka pengguna akan menerima request permissions apakah notifikasi bisa dikirimkan atau tidak
setelah akses diberikan makan Firebase akan mengirimkan token untuk mengidentifikasi perangkat pengguna. 
Pada sisi back end akan mengirimkan detail notifikasi yang diinginkan ke firebase. Kemudian firebase akan melakukan push notification.
5 Aplikasi juga telah ditambahkan konfigurasi CORS
