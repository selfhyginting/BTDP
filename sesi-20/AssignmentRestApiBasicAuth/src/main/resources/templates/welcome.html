We've got what you need!

You can easly hit the API with localhost:8084/items. Here some endpoints for you :

  • (GET) localhost:8084/items : for getting all items
  • (GET) localhost:8084/items/id : for getting one item by id
  • (POST) localhost:8084/items/ : for adding item on body request
  • (PUT) localhost:8084/items/id : for editing one item and put in into body request
  • (DELETE) localhost:8084/items/id : for deleting one item
