Panduan Penggunaan Aplikasi : 
1. Aplikasi dapat dijalankan terpisah sesuai dengan keinginan user. 
2. Compile terlebih dahulu file java kemudian jalankan class dengan nama yang sama pada file java.
3. Perhatikan pada bagian console untuk melihat hasil ataupun memasukkan inputan ke dalam program.
 
Terdapat 3 file java yaitu :
1. SelectionSorting.java : pengurutan data menggunakan algoritma Selection Sort. Ketika pengurutan berdasarkan ascending maka 
algoritma selection sort akan melakukan iterasi sebanyak data dikurangi 1 dan menyeleksi data yang terkecil. 
Kemudian data terkecil ditempatkan pada bagian shifted place karena sudah dicek pada semua data. 
Kemudian iterasi dilanjutkan. 

2. Binary.java : melakukan pencarian data yang telah disorting terlebih dahulu menggunakan algoritma Binary Search
dengan data statis yang telah dihardcode pada program.

3. Binary Search : melalukan pencarian data yang telah disorting terlebih dahulu menggunakan algoritma Binary Search dengan data yang dinamis 
sesuai dengan masukan dari pengguna.
