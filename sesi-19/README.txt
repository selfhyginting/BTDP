Panduan Penggunaan Aplikasi : 
1. Aplikasi API dapat dijalankan di lokal pengguna. IDE yang disarankan untuk menjalankan aplikasi adalah eclipse/netbeans.
2. Pastikan module dan dependensi yang diperlukan telah terinstall dalam komputer pengguna termasuk Enviroment Variable yang telah
di setting sebelumnya
3. Pada Dokumentasi API JavaDoc jalankan mvn javadoc:javadoc
4. Pada Dokumentasi API Swagger, ketika Aplikasi API telah dijalankan 
akses link berikut : 
	
	http://localhost:8080/swagger-ui.html#!/hello45controller/sayHelloUsingGET

5. Swagger akan menampilkan deskripsi api, method request yang tersedia beserta informasi endpoint yang tersedia
6. Pengguna dapat melakukan request dengan try it sesuai dengan ada atau tidaknya request body, param, header atau yang lainnya.
